# 🍽️ Sistema de Gestão de Cantina

Projecto académico desenvolvido em Java por um grupo de 5 estudantes.

## 📋 Descrição
Sistema que simula o funcionamento de uma cantina — consulta de cardápio,
criação de pedidos, controlo de estoque e geração de facturas.
